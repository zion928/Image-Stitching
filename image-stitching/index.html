이미지를 클릭하면 크기를 전환합니다

사용 방법

1. 스크린샷 촬영
화면을 스크롤하며 스크린샷을 여러 장 촬영합니다.

2. 업로드
촬영한 스크린샷을 드래그 앤 드롭, 파일 선택, 또는 Ctrl+V로 업로드합니다.
순서는 자동으로 정렬되므로 신경 쓰지 않아도 됩니다.

3. 병합
"병합" 버튼을 클릭합니다. 이미지 수에 따라 즉시~수십 초 소요됩니다.

4. 저장
결과를 확인하고 JPG/PNG 저장 또는 클립보드 복사를 합니다.

촬영 가이드

필수 조건

  • 인접 이미지 간 최소 2~3줄 이상 겹치도록 촬영해 주세요.
  • 같은 화면을 스크롤하며 촬영한 이미지여야 합니다.
  • 모든 이미지의 해상도(가로x세로)가 동일해야 합니다.
  • 최소 2장 이상, 최대 8장까지 권장합니다.

잘 되는 경우

  • 스크롤바가 보이는 화면 (게임 상세, 설정 등) — 자동 순서 정렬 가능
  • 고정 헤더/하단바가 있는 화면 (메신저, 게임 UI 등)
  • 전체 화면 캡처 (Alt+PrintScreen, 폰 기본 스크린샷)

잘 안 되는 경우

  • 수동으로 크롭하거나 편집한 이미지
  • 겹치는 영역이 전혀 없거나 1줄 미만인 경우
  • 서로 다른 페이지/화면의 스크린샷을 혼합한 경우
  • 스크롤 사이에 화면 내용이 변경된 경우 (실시간 피드 등)
