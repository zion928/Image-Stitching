# Auto Image Stitcher — 스크롤 캡처 자동 병합 웹 서비스

여러 장의 스크롤 캡처 이미지를 자동으로 이어붙여 한 장의 긴 이미지로 만들어 주는 **클라이언트 사이드 전용** 웹 서비스입니다.

## 주요 기능

| 기능 | 설명 |
|---|---|
| **다중 이미지 업로드** | 드래그 앤 드롭 또는 파일 선택으로 JPG/PNG/WebP 다중 업로드 |
| **드래그 앤 드롭 순서 변경** | 업로드된 이미지를 드래그하여 원하는 순서로 재배열 |
| **자동 순서 정렬** | 파일명/업로드 순서와 무관하게 겹치는 픽셀을 분석해 올바른 스크롤 순서를 자동 탐지 |
| **고정 UI 제거** | 상태바·앱 헤더·푸터 등 모든 캡처에 공통으로 나타나는 영역을 자동 감지 후 제거 |
| **Web Worker 처리** | 메인 UI를 멈추지 않고 백그라운드에서 병합 처리, 진행률 표시 |
| **결과 미리보기** | 클릭으로 원본 크기 토글 |
| **다운로드** | JPG(0.92품질) 또는 PNG 파일로 저장 |
| **클립보드 복사** | Clipboard API를 통해 결과 이미지를 클립보드에 복사 |
| **완전 클라이언트 사이드** | 서버로 이미지가 전송되지 않아 개인정보 유출 걱정 없음 |

## 사용 방법

1. `index.html`을 브라우저로 열거나, 저장소를 GitHub Pages로 배포합니다.
2. 캡처 이미지를 업로드합니다 (드래그 앤 드롭 또는 클릭).
3. 필요에 따라 이미지를 드래그하여 순서를 조정합니다.
4. 옵션 선택 후 **이미지 병합 시작** 버튼을 클릭합니다.
5. 결과 이미지를 PNG 또는 JPG로 저장하거나 클립보드에 복사합니다.

## 기술 스택

- **순수 HTML/CSS/JavaScript** — 외부 프레임워크 의존성 없음
- **Canvas API** — 이미지 픽셀 데이터 추출 및 렌더링
- **Web Worker API** — 백그라운드 스레드 처리
- **Clipboard API** — 클립보드 복사
- **HTML5 Drag and Drop API** — 이미지 순서 재배열

## 알고리즘 개요

### 겹침 탐지 (Overlap Detection)
각 이미지의 행(row)별 평균 RGB 시그니처를 계산한 뒤,
상단 이미지의 하단부와 하단 이미지의 상단부 사이의 **SAD(Sum of Absolute Differences)**를 계산합니다.
Coarse-to-Fine 2단계 검색으로 속도와 정확도를 모두 확보합니다.

### 자동 순서 정렬 (Auto Reorder)
모든 이미지 쌍에 대해 겹침 점수 매트릭스를 계산하고,
"가장 위에 올 이미지"를 휴리스틱으로 선택한 뒤 Greedy Nearest-Neighbour 방식으로 체인을 구성합니다.

### 고정 UI 제거 (Fixed UI Removal)
모든 이미지에서 동일 행의 평균 픽셀 값 차이가 임계값(12) 이하인 행을 상단/하단에서 연속으로 탐지하여 크롭합니다.
JPEG 압축 아티팩트를 허용 오차 내에서 처리합니다.

## 주의 사항

- 이미지 간 겹치는 영역이 너무 적으면 (< 5%) 단순 수직 스택으로 폴백됩니다.
- 모든 이미지의 너비가 동일한 것을 권장합니다 (같은 기기에서 캡처).
- 매우 많은 고해상도 이미지(20장 이상)는 브라우저 메모리 한계에 달할 수 있습니다.
